# AURORA - Winter '26 Edition

A stunning, immersive website experience designed to rival Shopify's Winter Edition announcement page. Built with pure HTML, CSS, and JavaScript for maximum performance and creative control.

## Features

### Visual Experience
- **Aurora Background Effects**: Dynamic, animated aurora borealis effect with multiple layered gradients
- **Starfield Animation**: Twinkling star particles for depth
- **Cursor Glow**: Interactive cursor following light effect
- **Scroll Progress Indicator**: Visual progress bar showing page position

### Interactive Elements
- **Smooth Scroll Animations**: Elements reveal as you scroll with staggered timing
- **Parallax Effects**: Multi-layer parallax for depth and immersion
- **Counter Animations**: Numbers animate up when scrolled into view
- **Carousel Showcase**: Touch-enabled, keyboard-navigable carousel
- **Magnetic Buttons**: Buttons subtly follow cursor movement
- **Typing Effect**: Auto-typing demonstration in feature section

### Design
- **Renaissance-meets-Modern Theme**: Inspired by Shopify's "RenAIssance" concept
- **Rich Color Palette**: Aurora-inspired gradient colors (purple, teal, pink, gold)
- **Typography**: Playfair Display for headings, Inter for body text
- **Responsive Design**: Fully responsive from mobile to 4K displays
- **Reduced Motion Support**: Respects user preferences for reduced motion

### Sections
1. **Hero**: Dramatic entrance with animated aurora background
2. **Vision**: Company philosophy with quote cards
3. **Features**: Grid layout showcasing product capabilities
4. **Showcase**: Interactive carousel demonstrating features
5. **Future**: Call-to-action with spinning ray effect
6. **Footer**: Full navigation footer

## Tech Stack

- **HTML5**: Semantic markup
- **CSS3**: Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript**: No dependencies, pure performance
- **Google Fonts**: Playfair Display, Inter, JetBrains Mono

## Getting Started

Simply open `index.html` in a modern browser. No build step required.

```bash
# Clone and open
git clone <repo-url>
cd WebBuilderFrontendExpirement
open index.html  # macOS
# or
xdg-open index.html  # Linux
# or just double-click index.html
```
